Web Framework for Crystal

Amber is a web application framework written in Crystal  inspired by a web of frameworks and other popular application frameworks.
Getting Started
Become a Contributor
Contributing to Amber can be a rewarding way to learn, teach, and build experience in just about any skill you can imagine.  Amber is a community effort and we want You to be part of ours

Fast. Cohesive. Efficient.

Amber makes building web applications simple, fast and enjoyable without sacrificing performance.

Build Quickly

Use code generators to and scaffolding to create rapid prototypes

Cohesive MVC

A framework with only what you need, no bloatware. Small footprint, little conventions, developer freedom.

No Configuration

No complicated XML, YAML just setup your ORM and you will be ready to shine

Feel at Home

Crystal’s syntax is heavily inspired on Ruby’s, so it feels natural to read and easy to write.

MIT License

Built by the community for everyone. MIT License == commercial ready.

High Performance

Compiled, blazingly fast, native code, low memory footprint. Engineers delight!
